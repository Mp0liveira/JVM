Descrição detalhada

Classe responsável por ler e interpretar os dados binários de um arquivo .class da JVM.

Implementa o padrão Singleton, contendo métodos para ler a estrutura de um arquivo .class, incluindo informações gerais, constant pool, métodos, campos e atributos.
